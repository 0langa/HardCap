# Support

Use GitHub Issues for bug reports, crashes, reproducible limit failures, and feature requests.

Before opening an issue, please include:

- HardCap version.
- Windows version and architecture.
- Whether HardCap was running elevated.
- The target application and whether it was already running or launched through HardCap.
- CPU and memory limits configured.
- Any warning or status text shown in the HardCap window.
